위에 대한 이슈

Git이 설치되어 있지 않아서 발생하는 에러입니다. Git을 설치해야 합니다:
1. Git 설치:
   - Git 공식 웹사이트에서 Windows용 Git 설치 프로그램을 다운로드
   - 다운로드한 설치 프로그램 실행
   - 기본 설정으로 설치 진행
2. 설치 완료 후 PowerShell이나 명령 프롬프트를 재시작
3. Git이 제대로 설치되었는지 확인:
git --version

4. 그 다음 Git 초기 설정:
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

5. 이제 Git 명령어를 실행할 수 있습니다:
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/kwo0two/rental-management.git
git push -u origin main
